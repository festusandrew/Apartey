# Apartey Agent Dashboard (Simplified)

## 🎯 Product Goal

A fast, clean dashboard that helps agents:

* Get more enquiries
* Respond quickly
* Track listing performance
* Close deals faster

---

# 🧭 OVERALL LAYOUT

### Sidebar (Left)

* Dashboard
* My Listings
* Enquiries
* Performance
* Profile
* Settings

---

# 🏠 1. DASHBOARD (HOME)

### Top Summary Cards

* Active Listings
* Total Enquiries (This Month)
* Response Rate
* Avg Response Time

---

### 🔥 Recent Enquiries

Simple list (not complex CRM):

* Name
* Property
* Time received
* Status tag (New / Contacted / Viewing / Closed)

Quick Actions:

* Call
* WhatsApp
* Mark as Contacted

---

### ⚡ Quick Actions Panel

* * Add New Listing
* Boost Listing
* View All Enquiries

---

# 🏢 2. MY LISTINGS

### Listing Cards

Each listing shows:

* Property image
* Title + location
* Price
* Status (Active / Expired)

### Performance Metrics (per listing)

* Views
* Clicks
* Enquiries

### Actions:

* Edit
* Boost
* Mark as Closed

---

### Filters

* Active
* Expired
* Most Viewed
* Most Enquiries

---

# 📩 3. ENQUIRIES (LEAD MANAGEMENT)

### Pipeline (Simple)

* New
* Contacted
* Viewing Scheduled
* Closed

Each lead card contains:

* Name
* Phone
* Property of interest
* Time received
* Notes (optional)

### Quick Actions:

* Call
* WhatsApp
* Move stage

---

# 📊 4. PERFORMANCE

### Overview Metrics

* Total Listings
* Total Enquiries
* Conversion Rate (Closed / Enquiries)
* Avg Response Time

---

### Listing Performance Table

Columns:

* Property
* Views
* Clicks
* Enquiries
* Conversion

---

### Insights (Simple Text)

* "Your 2-bedroom in Wuse is getting the most interest"
* "Listings with 5+ photos get 2x more enquiries"

---

# 👤 5. PROFILE (PUBLIC VIEW CONTROL)

Editable Fields:

* Name
* Title (e.g., Property Consultant)
* Profile photo
* Agency/Business name
* Years of experience
* Languages
* Regions
* Bio (max 600 chars)

### Trust Indicators

* Verified Badge (if applicable)
* Response Time Badge
* Total Listings
* Successful Matches

---

# ⚙️ 6. SETTINGS

* Notification preferences
* Contact visibility (phone/email toggle)

---

# 🔔 NOTIFICATIONS SYSTEM

Smart notifications only:

* New enquiry
* Missed response (after X mins)
* Listing expiring soon

---

# 🎨 UX PRINCIPLES

* Mobile-first (most agents use phones)
* Large tap targets
* Minimal typing
* Fast actions (Call, WhatsApp first)
* No complex CRM feel

---

# 🔥 CORE DIFFERENCE

This is NOT a full CRM.
This is a:
👉 High-speed listing + enquiry conversion tool
